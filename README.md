# BarPress: Impulse Ascension

A modular incremental browser game inspired by Spacebar Clicker, redesigned with a futuristic economy, prestige layers, offline progress, achievements, audio hooks, responsive UI, and canvas-based effects.

## Run

Open `index.html` in a modern browser.

## Features

- Spacebar, tap, hold, rhythm, criticals, and anti-spam balancing
- Manual upgrades, automation, synergy, quantum, cosmic, and transcendence layers
- Prestige with Shards and Relics
- Save/load with versioning and corruption checks
- Offline progression
- Particles, floating numbers, screen shake
- Audio synthesis hooks
- Huge-number formatting
- Achievements and hidden secrets

## Notes

The game is framework-free and uses native ES modules.
